# KryonOS Official Developer Repositories

Welcome to the official KryonOS developer repositories.  

> ⚠️ **Important:** These repositories are intended **only for developers** who want to build KryonOS from source.

If you are a developer and want to compile KryonOS yourself, please visit our **main repository** to find detailed instructions on how to set up your environment and build the ROM:  

[Main KryonOS Repository](https://github.com/KryonOS/main)  

For regular users who just want to use KryonOS, you can download the official builds directly from our website:  

[KryonOS.net Downloads](https://www.kryonos.net)  

---

**Notes for Developers:**  
- These repos contain source code and manifests for building KryonOS.  
- Forking, contributing, or modifying the ROM should follow the guidelines in the main repository.  
- Be careful: using these repos incorrectly may lead to incomplete or broken builds.  

Thank you for your interest in KryonOS!
